# SandraImmobiliere

SandraImmobiliere is a modern real estate management application built with the MERN stack (MongoDB, Express, React, Node.js). It allows real estate agents and agencies to manage properties, clients, and transactions efficiently in a single platform. The application is fully responsive, providing a seamless experience on both web and mobile devices.

## Key Features
- User authentication and role management
- Property listings with details and images
- Client management with contact information and history
- Transaction and sales tracking
- Responsive and intuitive user interface

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Version Control:** Git & GitHub

## Purpose
This project demonstrates a full-stack web application for the real estate industry, focusing on usability, scalability, and modern web development practices.
